# Django CMS Taxonomy

Taxonomy support for django CMS.

## Features

- Manage taxonomies and terms in django CMS
- REST API support
- Multi-language support
- Easy integration with existing CMS

## Installation

Install using pip:

\`\`\`bash
pip install djangocms-taxonomy
\`\`\`

Update your \`INSTALLED_APPS\` setting:

\`\`\`python
INSTALLED_APPS = [
    ...
    \"djangocms_taxonomy\",
    ...
]
\`\`\`
